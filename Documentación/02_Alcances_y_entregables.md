# Alcances y Entregables

## Alcance del Proyecto

En su Primera Etapa (MVP), el proyecto abarcará el desarrollo de una Aplicación Web Progresiva (PWA) optimizada principalmente para dispositivos móviles y uso en navegador, dejando fuera el desarrollo de aplicaciones de escritorio nativas.

### Funcionalidades Core (MVP)

1. **Gestión de Viajes y Usuarios**
   - Creación de nuevos viajes.
   - Sistema de invitación de participantes mediante enlaces o invitaciones directas.

2. **Toma de Decisiones (Encuestas)**
   - Creación de encuestas para definir variables del viaje (destinos, fechas, actividades).
   - Posibilidad de que los invitados sugieran opciones.
   - Configuración de fechas límite (deadlines) de votación.

3. **Gestión de Presupuesto y Finanzas**
   - Registro de costos/gastos aproximados (alquiler, pasajes, combustible, comida).
   - Sistema de división asimétrica de gastos (asignar porcentajes o montos específicos por persona, manejo de deudas cruzadas).

4. **Logística y Transporte**
   - Registro de medios de transporte disponibles (quién lleva auto).
   - Asignación de pasajeros a cada vehículo.
   - Lista colaborativa de objetos/equipaje y asignación de responsables (qué lleva cada persona y en qué vehículo va).

## Entregables Principales

- **Documento de Requerimientos de Software (SRD):** Documentación técnica detallada (Arquitectura, APIs, Modelos de datos).
- **MVP Funcional:** Aplicación web PWA desplegada y accesible, respetando el stack tecnológico base (Next.js, Tailwind, React Query, Zustand).
- **Documentación del Proyecto:** Carpeta organizativa estandarizada con objetivos, responsables, cronograma y reportes.

## Fuera de Alcance Inicial

- Aplicaciones Desktop nativas (Electron/Tauri).
- Sistema de reservas de vuelos/hoteles integrado con agencias externas.
- Pasarelas de pago internas (los pagos/transferencias se acuerdan fuera de la app).
